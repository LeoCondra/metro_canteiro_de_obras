# Controle de Canteiro de Obra
Um aplicativo para acompanhar e gerenciar atividades de uma construção civil.

---

## Tecnologias Utilizadas
- **Frontend**: [React](https://react.dev/) (JavaScript)
- **Backend**: API do **Google Vision** integrada via *Edge Function* no [Supabase](https://supabase.com/)
- **Armazenamento**: *Storage Bucket* no [Supabase](https://supabase.com/) para arquivos BIM e imagens

---

##  Como rodar o projeto

1. Abra um terminal **bash**
2. Entre no repositório do projeto
  `cd metro_canteiro_de_obras`
3. Instale as dependências:
   `npm install`
4. Execute o servidor de desenvolvimento:
   `npm run dev`
